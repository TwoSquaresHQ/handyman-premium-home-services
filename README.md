## Deploy On Vercel

1. Install dependencies:

```bash
pnpm install
```

2. Run locally:

```bash
pnpm dev
```

3. Deploy to Vercel:

- import the repo into Vercel
- set `NEXT_PUBLIC_SITE_URL`
- deploy

Or use the Deploy with Vercel button above for a one-click setup flow.
